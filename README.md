## 📦 部署

推荐使用 Gitee Pages 或 Cloudflare Pages：

```bash
# 初始化 Git
git init
git add .
git commit -m "Initial commit"

# 推送到 Gitee
git remote add origin https://gitee.com/zpdr/test.git
git push -u origin main
```

然后在 Gitee 仓库设置中开启 Gitee Pages。
